###Requirements

Python v3 and beyond

###To run

python SudokuSolver.py target_puzzle.txt Algorithm

eg. python SudokuSolver.py puzzle1.txt BF

#######Algorithms

BF = Brute Force
FC-MRV = Forward Checking (Minimum remaining values)
BT = Backtrack
